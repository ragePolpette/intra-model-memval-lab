# GPT-2 Small Notes

Questa cartella raccoglie note operative per la prima verticale concreta GPT-2 small.

- `datasets/`: dataset sintetico seedato e leggibile.
- `prompts/`: template testuali per trace ed evaluation.
- `configs/`: configurazioni minime riusabili dalla CLI.
- `runs/`: destinazione logica per output esportati o snapshot manuali, non usata come storage primario.

Gli artifact persistenti reali restano nello store configurato via CLI (`--db-path`, `--artifact-dir`).
